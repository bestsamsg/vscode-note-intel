# note-intel
提供源码修改记录注释模板

## Features
可以输入以下内容，并结合ctrl+enter组合键生成注释模板
```
//note-add
//note-add-begin

//note-rem
//note-remove
//note-remove-begin

//note-rep
//note-replace
//note-replace-begin
```


## Extension Settings

用于定义注释前缀
```
note-intel.prefix
```

For example:

This extension contributes the following settings:

* `note-intel.prefix`: 'note'
